05

插画 / Plant Illustrations

手账水彩风格:1.5px 描线 + 半透明色块 + 纸纹叠加。V1 必备 3 个变体(normal / thirsty / just-watered)。

normal · 默认
翠绿,直立,缓摇 ±2° / 2400ms
thirsty · 缺水
灰绿,微垂,急促摇 ±4° / 1800ms
just-watered · 刚浇过
深绿,叶尖暖光,leaf-lift 弹起
